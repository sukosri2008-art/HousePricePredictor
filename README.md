# HousePricePredictor
 A Streamlit web app for predicting house prices using Machine Learning.

A Machine Learning project with an interactive web interface built using Streamlit.

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🌐 Features
- Interactive UI with Streamlit
- Real-time predictions
- Easy-to-use interface
